To paraphrase the doc : ```_A portable volatile memory acquisition tool for Linux._

AVML is an X86_64 userland volatile memory acquisition tool written in [Rust](https://www.rust-lang.org/), intended to be deployed as a static binary. AVML can be used to acquire memory without knowing the target OS distribution or kernel a priori. No on-target compilation or fingerprinting is needed.```

The documentation is so clean that I highly encourage you to check it here : https://github.com/microsoft/avml instead of a reading copy-pasta version of it
